# Tailwind CSS Checkbox - React

Use our Tailwind CSS `Checkbox` to allow the user to select one or more items from a set.

You can use a `Checkbox` for:

- Selecting one or more options from a list.
- Presenting a list containing sub-selections.
- Turning an item on/off in a desktop environment (If you have a single option, avoid using a `Checkbox` and use an on/off switch instead).

See below our simple `Checkbox` example that you can use in your React and Tailwind CSS projects. The example also comes in different colors, so you can adapt it easily to your needs.

## Checkbox Colors

The `Checkbox` component comes with 6 different colors that you can change it using the `color` prop.

<ComponentPreview demo={<Checkbox.CheckboxColors />}>
```tsx

  return (
    <div className="flex flex-wrap justify-center gap-4">

      <Checkbox color="secondary" />

      <Checkbox color="success" />
      <Checkbox color="warning" />
      <Checkbox color="error" />
    </div>
  );
}
```
</ComponentPreview>

## Checkbox With Label

You can use the `Typography` component to add a label to your `Checkbox`.

<ComponentPreview demo={<Checkbox.CheckboxWithLabel />}>
```tsx
import { Checkbox, Typography } from "@material-tailwind/react";

const ID = "checkbox";

export function CheckboxWithLabel() {
  return (
    <div className="flex items-center gap-2">
      <Checkbox id={ID} />
      <Typography
        as="label"
        htmlFor={ID}
        className="text-gray-500 dark:text-gray-400"
      >
        Remember Me
      </Typography>
    </div>
  );
}
```
</ComponentPreview>

## Custom Checkbox

You can customize the `Checkbox` using the `baseClassName`, `className` and `iconClassName` props.

<ComponentPreview demo={<Checkbox.CustomCheckbox />}>
```tsx
import { Checkbox } from "@material-tailwind/react";

export function CustomCheckbox() {
  return (
    <Checkbox
      iconClassName="scale-50 peer-checked:scale-75"
      className="rounded-full bg-gray-200 dark:bg-gray-900"
    />
  );
}
```
</ComponentPreview>

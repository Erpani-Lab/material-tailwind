# Tailwind CSS Pagination - React

Use our `Pagination` based on Tailwind CSS for actions in forms, dialogues, and more.

`Pagination` is an essential element of web design. Basically,
`Pagination` is styled links that grab the user's attention. They help users
navigate our websites or apps and drive them to a particular action like submitting
a contact form or placing an order as easy as possible.

See below our `Pagination` example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

## Simple Pagination

<ComponentPreview demo={<Pagination.SimplePagination />}>
```tsx
import { Button, IconButton, Typography } from "@material-tailwind/react";
import { NavArrowRight, NavArrowLeft } from "iconoir-react";

export function SimplePagination() {
  return (
    <div className="flex items-center gap-1">
      <IconButton variant="ghost">
        <NavArrowLeft className="h-4 w-4 stroke-2" />
      </IconButton>
      <Typography className="text-gray-600 dark:text-gray-400">
        Page{" "}
        <Typography as="stronge" color="primary">
          2
        </Typography>{" "}
        of{" "}
        <Typography as="stronge" color="primary">
          10
        </Typography>
      </Typography>
      <IconButton variant="ghost">
        <NavArrowRight className="h-4 w-4 stroke-2" />
      </IconButton>
    </div>
  );
}
```
</ComponentPreview>

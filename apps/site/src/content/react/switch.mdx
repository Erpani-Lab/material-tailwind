# Tailwind CSS Checkbox - React

Use our Tailwind CSS `Checkbox` to allow the user to select one or more items from a set.

You can use a `Checkbox` for:

- Selecting one or more options from a list.
- Presenting a list containing sub-selections.
- Turning an item on/off in a desktop environment (If you have a single option, avoid using a `Checkbox` and use an on/off switch instead).

See below our simple `Checkbox` example that you can use in your React and Tailwind CSS projects. The example also comes in different colors, so you can adapt it easily to your needs.

## Switch Colors

The `Switch` component comes with 6 different colors that you can change it using the `color` prop.

  return (

      <Switch color="warning" />
      <Switch color="error" />
    </div>
  );

## Switch With Label

You can use the `Typography` component to add a label to your `Switch`.

<ComponentPreview demo={<Switch.SwitchWithLabel />}>
```tsx
import { Switch, Typography } from "@material-tailwind/react";

const ID = "switch";

export function SwitchWithLabel() {
  return (
    <div className="flex items-center gap-2">
      <Switch id={ID} />
      <Typography
        as="label"
        htmlFor={ID}
        className="text-gray-500 dark:text-gray-400"
      >
        Dark Mode
      </Typography>
    </div>
  );
}
```
</ComponentPreview>

# Tailwind CSS Card - React

Use our `Card` based on Tailwind CSS for actions in forms, dialogues, and more.

`Card` is an essential element of web design. Basically,
`Card` is styled links that grab the user's attention. They help users
navigate our websites or apps and drive them to a particular action like submitting
a contact form or placing an order as easy as possible.

See below our `Card` example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

  Card,
  Typography,
  Button,
  Input,
  Checkbox,
} from "@material-tailwind/react";

    <Card className="max-w-xs">
      <Card.Header
        as={Card}
        color="primary"
        className="grid h-24 place-items-center shadow-none"

        <Typography as="span" variant="h4" color="secondary">
          Sign In
        </Typography>
      </Card.Header>
      <Card.Body as="form">
        <Input isFullWidth className="mb-4 mt-2 space-y-1.5">

            as="label"
            htmlFor="email"
            variant="small"
            color="primary"
            className="font-semibold"

            id="email"
            type="email"

# Tailwind CSS Typography - React

Use our Tailwind CSS `Typography` to present your website/web app as clearly and efficiently as possible.

Below we are presenting a simple `Typography` component example. It comes in different styles and colors, so you can adapt it easily to your needs.

---

## Headings

<ComponentPreview demo={<Typography.Headings />}>
```tsx
import { Typography } from "@material-tailwind/react";

export function Headings() {
  return (
    <div className="flex flex-col gap-4 py-4">
      <Typography variant="h1">Material Tailwind (H1)</Typography>
      <Typography variant="h2">Material Tailwind (H2)</Typography>
      <Typography variant="h3">Material Tailwind (H3)</Typography>
      <Typography variant="h4">Material Tailwind (H4)</Typography>
      <Typography variant="h5">Material Tailwind (H5)</Typography>
      <Typography variant="h6">Material Tailwind (H6)</Typography>
    </div>
  );
}

```
</ComponentPreview>

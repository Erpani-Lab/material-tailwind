# Tailwind CSS List - React

Easily create `List` with different statuses and sizes using our components based on Tailwind CSS and React.

`List` fields are an essential user interface design element, providing users with the means to enter non-standardized responses.

See below our `List` component examples. It comes in different styles and colors, so you can adapt it easily to your needs.

## List with Avatar

<ComponentPreview demo={<List.ListWithAvatar />}>
```tsx
import * as React from "react";
import { List, Avatar, Typography } from "@material-tailwind/react";

const ListItem = React.forwardRef(
  ({ img, title, description, ...rest }, ref) => {
    return (
      <List.Item ref={ref} {...rest}>
        <List.ItemStart>
          <Avatar src={img} alt="profile-picture" />
        </List.ItemStart>
        <div>
          <Typography className="font-semibold">{title}</Typography>
          <Typography
            type="small"
            className="text-gray-600"
          >
            {description}
          </Typography>
        </div>
      </List.Item>
    );
  },
);

export function ListWithAvatar() {
```

## List with Selected Item

<ComponentPreview demo={<List.ListWithSelectedItem />}>
```tsx
import * as React from "react";
import { List } from "@material-tailwind/react";

export function ListWithSelectedItem() {
  const [selected, setSelected] = React.useState("inbox");

  return (
    <List>
      <List.Item
        selected={selected === "inbox"}
        onClick={() => setSelected("inbox")}
      >
        Inbox
      </List.Item>
      <List.Item
        selected={selected === "trash"}
        onClick={() => setSelected("trash")}
      >
        Trash
      </List.Item>
      <List.Item
        selected={selected === "settings"}
        onClick={() => setSelected("settings")}
      >
        Settings
      </List.Item>
    </List>
  );
}
```
</ComponentPreview>

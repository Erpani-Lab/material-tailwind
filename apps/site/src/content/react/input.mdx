# Tailwind CSS Input - React

Easily create `Input` with different statuses and sizes using our components based on Tailwind CSS and React.

`Input` fields are an essential user interface design element, providing users with the means to enter non-standardized responses.

See below our `Input` component examples. It comes in different styles and colors, so you can adapt it easily to your needs.

---

## Input Demo

<ComponentPreview demo={<Input.InputDemo />}>
```tsx
import { Input } from "@material-tailwind/react";

export function InputDemo() {
  return (
    <Input>
      <Input.Field placeholder="Input" />
    </Input>
  );
}
```
</ComponentPreview>

## Input Sizes

The `Input` component comes with 3 different sizes that you can change it using the `size` prop.

<ComponentPreview demo={<Input.InputSizes />}>
```tsx
import { Input } from "@material-tailwind/react";

export function InputSizes() {
  return (
    <div className="w-[240px] space-y-4">
      <Input size="sm">
        <Input.Field placeholder="Input Small" />
      </Input>
      <Input size="md">
        <Input.Field placeholder="Input Medium" />
      </Input>
      <Input size="lg">
        <Input.Field placeholder="Input Large" />
      </Input>
    </div>
  );
}
```
</ComponentPreview>

## Input with Validation

You can combine the `Input` component with the `Typography` component to create a validation message, use the `isSuccess` prop for the success state and the `isError` prop for the error state.

<ComponentPreview demo={<Input.InputWithValidation />}>
```tsx
import { Input, Typography } from "@material-tailwind/react";

export function InputWithValidation() {
  return (
    <div className="w-[240px] space-y-6">
      <Input isError color="error" className="w-full space-y-1">
        <Input.Field placeholder="Input Error" />
        <Typography variant="small" color="error">
          Something went wrong!
        </Typography>
      </Input>
      <Input isSuccess color="success" className="w-full space-y-1">
        <Input.Field placeholder="Input Success" />
        <Typography variant="small" color="success">
          Congratulations ðŸŽ‰
        </Typography>
      </Input>
    </div>
  );
}
```
</ComponentPreview>

## Input with Icon

You can add an icon for the `Input` component using the `Input.Icon` component.

The below example is using [Iconoir](https://iconoir.com?ref=material-tailwind) for the icons.

<ComponentPreview demo={<Input.InputWithIcon />}>
```tsx
import { Input } from "@material-tailwind/react";
import { ProfileCircle, Lock } from "iconoir-react";

export function InputWithIcon() {
  return (
    <div className="w-[240px] space-y-4">
      <Input>
        <Input.Field placeholder="Username" />
        <Input.Icon>
          <ProfileCircle className="h-full w-full" />
        </Input.Icon>
      </Input>
      <Input>
        <Input.Field type="password" placeholder="Password" />
        <Input.Icon placement="end">
          <Lock className="h-full w-full" />
        </Input.Icon>
      </Input>
    </div>
  );
}
```
</ComponentPreview>

## Input with Label

You can combine the `Input` component with the `Typography` component to add a label for the `Input` component.

<ComponentPreview demo={<Input.InputWithLabel />}>
```tsx
import { Input, Typography } from "@material-tailwind/react";

export function InputWithLabel() {
  return (
    <Input className="w-[240px] space-y-1.5">
      <Typography
        as="label"
        htmlFor="email"
        variant="small"
        color="primary"
        className="font-medium"
      >
        Email
      </Typography>
      <Input.Field id="email" type="email" placeholder="someone@example.com" />
    </Input>
  );
}
```
</ComponentPreview>

## Input with Helper Text

You can combine the `Input` component with the `Typography` component to add some description under the `Input` component.

The below example is using [Iconoir](https://iconoir.com?ref=material-tailwind) for the icons.

<ComponentPreview demo={<Input.InputWithHelperText />}>
```tsx
import { Input, Typography } from "@material-tailwind/react";
import { InfoCircle } from "iconoir-react";

export function InputWithHelperText() {
  return (
    <Input className="w-[240px] space-y-2">
      <Input.Field type="password" placeholder="Password" />
      <div className="flex gap-1.5 text-gray-600">
        <InfoCircle className="h-3.5 w-3.5 shrink-0 translate-y-[3px] stroke-2" />
        <Typography variant="small">
          Use at least 8 characters, one uppercase, one lowercase and one
          number.
        </Typography>
      </div>
    </Input>
  );
}
```
</ComponentPreview>

## Input with Button

You can combine the `Input` component with the `Button` component for creating forms.

<ComponentPreview demo={<Input.InputWithButton />}>
```tsx
import { Button, Input } from "@material-tailwind/react";

export function InputWithButton() {
  return (
    <form action="#" className="flex w-full items-center justify-center gap-2">
      <Input>
        <Input.Field type="email" placeholder="someone@example.com" />
      </Input>
      <Button type="submit">Subscribe</Button>
    </form>
  );
}
```
</ComponentPreview>

## Input Custom Styles

Each piece of `Input` component is customizable using the `className` prop.

<ComponentPreview demo={<Input.InputCustomStyles />}>
```tsx
import { Input } from "@material-tailwind/react";
import { ProfileCircle } from "iconoir-react";

export function InputCustomStyles() {
  return (
    <Input>
      <Input.Field
        placeholder="Username"
        className="rounded-none border-0 border-b border-gray-400 pr-0.5 ring-0 hover:border-gray-900 focus:border-gray-900 data-[icon-placement=start]:pl-[26px] dark:border-gray-600 dark:hover:border-gray-50 dark:focus:border-gray-50"
      />
      <Input.Icon className="data-[placement=start]:left-px">
        <ProfileCircle className="h-full w-full" />
      </Input.Icon>
    </Input>
  );
}
```
</ComponentPreview>

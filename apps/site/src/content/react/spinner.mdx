# Tailwind CSS Spinner - React

Use our `Spinner` based on Tailwind CSS for actions in forms, dialogues, and more.

`Spinner` is an essential element of web design. Basically,
`Spinner` is styled links that grab the user's attention. They help users
navigate our websites or apps and drive them to a particular action like submitting
a contact form or placing an order as easy as possible.

See below our `Spinner` example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

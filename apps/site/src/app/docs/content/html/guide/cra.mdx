# Integrating @material-tailwind/html with Create React App: A Comprehensive Guide

In this guide, we'll walk you through the process of integrating the `@material-tailwind/html` library into a Create React App project. This integration allows you to harness the power of Material Design components within your React applications, ensuring a sleek and responsive user interface.

<br></br><br></br>

## Prerequisites

Before we begin, ensure you have the following installed:

- **Node.js**: Version 14 or later.
- **npm**: Version 6 or later.

If you haven't installed these yet, download Node.js from the [official Node.js website](https://nodejs.org/).

<br></br><br></br>

## Step 1: Setting Up a New Create React App Project

Start by creating a new React project using Create React App. Open your terminal and run:

<CodePreviewTailwindClasses codeSnippet={`npx create-react-app my-material-tailwind-app
cd my-material-tailwind-app`}/>

<br></br><br></br>

## Step 2: Installing Tailwind CSS

Install Tailwind CSS and its peer dependencies:

<CodePreviewTailwindClasses codeSnippet={`npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}/>

<br></br>

Configure your `tailwind.config.js` or `tailwind.config.ts`:

<CodePreviewTailwindClasses codeSnippet={`// For JavaScript (tailwind.config.js)
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
 
    // Or if using \`src\` directory:
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

// For TypeScript (tailwind.config.ts)
import type { Config } from 'tailwindcss'

export default {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./pages/**/*.{js,ts,jsx,tsx}",
    "./components/**/*.{js,ts,jsx,tsx}",
 
    // Or if using \`src\` directory:
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
} satisfies Config`}/>

<br></br>

Add Tailwind directives to your `src/index.css`:

<CodePreviewTailwindClasses codeSnippet={`@tailwind base;
@tailwind components;
@tailwind utilities;`}/>

<br></br><br></br>

## Step 3: Installing @material-tailwind/html

<CodePreviewTailwindClasses codeSnippet={`npm install @material-tailwind/html`}/>

<br></br><br></br>

## TypeScript Configuration

If you're using TypeScript (create-react-app --template typescript), ensure your `tsconfig.json` includes:

<CodePreviewTailwindClasses codeSnippet={`{
  "compilerOptions": {
    "moduleResolution": "node",
  }
}`}/>

<br></br>

> **Note**: Create React App comes with TypeScript support out of the box when using the TypeScript template. You can create a new TypeScript project using `npx create-react-app my-app --template typescript`.

<br></br><br></br>

## Building for Production

When you're ready to build your React app for production:

<CodePreviewTailwindClasses codeSnippet={`npm run build`}/>

<br></br>

This will create an optimized production build in the `build` directory.

# Integrating @material-tailwind/html with Astro: A Comprehensive Guide

In this guide, we'll walk you through the process of integrating the `@material-tailwind/html` library into an Astro project. This integration allows you to harness the power of Material Design components within your Astro applications, ensuring a sleek and responsive user interface.

<br></br><br></br>

## Prerequisites

Before we begin, ensure you have the following installed:

- **Node.js**: Version 16 or later.
- **npm**: Version 7 or later.

If you haven't installed these yet, download them from the [official Node.js website](https://nodejs.org/).

<br></br><br></br>

## Step 1: Setting Up a New Astro Project

Start by creating a new Astro project. Open your terminal and run:

<CodePreviewTailwindClasses codeSnippet={`npm create astro@latest my-material-tailwind-app
cd my-material-tailwind-app
npm install`}/>

<br></br><br></br>

## Step 2: Installing Tailwind CSS

Install Tailwind CSS and its dependencies:

<CodePreviewTailwindClasses codeSnippet={`npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}/>

<br></br>

Add Tailwind directives in `src/styles/global.css`:

<CodePreviewTailwindClasses codeSnippet={`@tailwind base;
@tailwind components;
@tailwind utilities;`}/>

<br></br>

Modify `tailwind.config.cjs` or `tailwind.config.ts`:

<CodePreviewTailwindClasses codeSnippet={`/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{astro,js,jsx,ts,tsx,vue,svelte}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};`}/>

<br></br><br></br>

## Step 3: Installing @material-tailwind/html

<CodePreviewTailwindClasses codeSnippet={`npm install @material-tailwind/html`}/>

<br></br><br></br>

## TypeScript Example

For more control and type safety, you can use the programmatic approach with full TypeScript support in your Astro project:

<CodePreviewTailwindClasses codeSnippet={`import { useEffect } from 'react';
import { Accordion } from "@material-tailwind/html";
import type { AccordionConfig, IAccordion } from "@material-tailwind/html";

function TypeScriptExample() {
  useEffect(() => {
    const container = document.getElementById('accordion-container');

    if (container) {
      const config: AccordionConfig = {
        exclusive: true,
        allOpen: false,
      };
      const accordion: IAccordion = new Accordion(container, config);
    }
  }, []);

  return (
    <div id="accordion-container" className="group block w-full">
      <div className="flex items-center justify-between w-full py-5 text-left font-medium text-gray-800 cursor-pointer" aria-expanded="false" id="button-1">
        Material Tailwind Astro
        <svg data-accordion-icon width="1.5em" height="1.5em" strokeWidth="1.5" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg" color="currentColor" className="h-4 w-4 transition-transform duration-300 rotate-180">
          <path d="M6 9L12 15L18 9" stroke="currentColor" strokeLinecap="round" strokeLinejoin="round"></path>
        </svg>
      </div>
      <div id="button-1" className="overflow-hidden transition-all duration-300 border-b border-gray-200">
        <p className="mb-5 text-sm text-gray-600">Material Tailwind is an open-source library crafted with Tailwind CSS. Get Material Tailwind and take advantage of its free components and features.</p>
      </div>
    </div>
  );
}

export default TypeScriptExample;`}/>

<br></br>

> **Note**: This TypeScript example demonstrates full type safety and programmatic control of the accordion component.

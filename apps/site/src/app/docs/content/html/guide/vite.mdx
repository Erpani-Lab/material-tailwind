# Integrating @material-tailwind/html with Vite + React: A Comprehensive Guide

In this guide, we'll walk you through the process of integrating the `@material-tailwind/html` library into a Vite + React project. This integration allows you to harness the power of Material Design components within your Vite applications, ensuring a sleek and responsive user interface. For other Vite project types (Vue, Svelte, Vanilla etc.), please refer to the [official Vite documentation](https://vitejs.dev/guide/).

<br></br><br></br>

## Prerequisites

Before we begin, ensure you have the following installed:

- **Node.js**: Version 12 or later.
- **npm**: Version 6 or later.

If you haven't installed these yet, download them from the [official Node.js website](https://nodejs.org/).

<br></br><br></br>

## Step 1: Setting Up a New Vite + React Project

Start by creating a new Vite project with React template. Open your terminal and run:

<CodePreviewTailwindClasses codeSnippet={`npm create vite@latest my-material-tailwind-app -- --template react
cd my-material-tailwind-app
npm install`}/>

<br></br><br></br>

## Step 2: Installing Tailwind CSS

Install Tailwind CSS and its dependencies:

<CodePreviewTailwindClasses codeSnippet={`npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}/>

<br></br>

Add Tailwind directives in `src/index.css`:

<CodePreviewTailwindClasses codeSnippet={`@tailwind base;
@tailwind components;
@tailwind utilities;`}/>

<br></br>

Configure your `tailwind.config.js` or `tailwind.config.ts`:

<CodePreviewTailwindClasses codeSnippet={`// For JavaScript (tailwind.config.js)
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './index.html',
    './src/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};

// For TypeScript (tailwind.config.ts)
import type { Config } from 'tailwindcss'

export default {
  content: [
    './index.html',
    './src/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
} satisfies Config`}/>

<br></br><br></br>

## Step 3: Installing @material-tailwind/html

<CodePreviewTailwindClasses codeSnippet={`npm install @material-tailwind/html`}/>

<br></br><br></br>

## TypeScript Configuration
If you're using TypeScript and encounter type resolution issues, ensure your `vite.config.ts` and `tsconfig.json` are properly configured:

<CodePreviewTailwindClasses codeSnippet={`// vite.config.ts
import { defineConfig } from "vite";
import checker from "vite-plugin-checker";

export default defineConfig({
  plugins: [
    checker({ typescript: true }),
  ],
  root: "./src",
  build: {
    outDir: "../dist",
  },
});


// tsconfig.json
{
  "compilerOptions": {
    "moduleResolution": "node",
  }
}`}/>

<br></br> 

> **Note**: The library's type definitions require Node.js resolution mode to work properly. Vite's default ESM configuration works seamlessly with Material Tailwind's TypeScript definitions.
